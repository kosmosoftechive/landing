## Deployment Steps (Should run from personal github, not company github)

1. **Install node modules from project folder**

    ```bash
    npm install
    ```

2. **Build production version**

    ```bash
    npm run export
    ```

3. **Deploy!**

    ```bash
    npm run deploy
    ```
